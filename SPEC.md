# personal-pm 需求规格书

> 本文件是所有讨论的唯一依据。任何偏离本文档的选型/实现都视为跑偏。

## 用户画像

- 主业绘画，副业编程
- ADHD + 阿斯伯格 — 启动困难，需要外部推动
- 工具必须减少工作量，不能增加维护负担
- 输入方式：微信发一句话就够了
- 查看方式：电脑上能看，手机上能看，打开就知道接下来做什么

## 核心需求

### 1. 远程任务录入

微信发一句话 → AI 拆解 → 出现在我的日程里。

```
"下周五之前交稿，今天练一小时人体，下午修两个bug"
                    ↓
           AI 拆成具体任务 + 安排时间
                    ↓
           "10:00-11:00 人体练习"
           "14:00-16:00 修登录bug"
           "16:00-17:00 修样式bug"
```

我不需要自己拖任务、分标签、填时间。AI 帮我做好。

### 2. 时间感知

任务能带时间。不只是 Todo List，是"几点做什么"。

"晚上六点理发" → 出现在今天 18:00。

### 3. 外部监督

- 到时间了提醒我
- 走神了提醒我
- 完成了给我反馈
- 我不需要主动去检查"接下来做什么"

### 4. 跨设备

- Windows 电脑上能看（主力工作机）
- iPad 上能看
- 数据同步自动，不需要手动操作

### 5. 活动追踪（锦上添花）

- 自动记录在用什么软件
- 走神检测（刷 B 站超时提醒）
- 工时自动统计

## 不需要的东西

- 精细的项目管理（标签、优先级、预估工时）
- 手动拖拽排期
- 复杂的分类体系
- 团队协作功能

## SP 能做什么 / 不能做什么

| 能力 | SP 能 | 说明 |
|---|---|---|
| 本地任务管理 | ✅ | 创建、完成、计时 |
| 跨设备同步 | ✅ | SuperSync |
| 远程 API | ❌ | 只有本地 3876 端口 |
| 服务端运行 | ❌ | 需要 GUI，无法 headless |
| 插件 | 🟡 | 仅桌面版，不能远程触发 |
| AI 集成 | ❌ | 无 |
| 自定义提醒 | ❌ | 无 |
| GitHub Issues 同步 | ✅ | 内置集成 |
| Google 日历同步 | ✅ | 内置集成 |

## 候选方案

### Vikunja（替代 SP）

- 自托管，有完整 REST API（我能远程操控）
- 支持 API Token 认证
- 有 Web 界面 + 移动端
- 支持标签、到期日、优先级
- 有 webhook 和提醒（邮件）
- 社区活跃，2025-2026 持续更新

**SP vs Vikunja 对比：**

| | SP | Vikunja |
|---|---|---|
| 我能远程创建任务 | ❌ | ✅ REST API |
| 任务带时间 | 🟡 日历同步 | ✅ 内置到期日+提醒 |
| 跨设备 | ✅ SuperSync | ✅ Web + 移动端 |
| 自托管 | ❌ 仅桌面 | ✅ Docker |
| 活动追踪 | ❌ | ❌（需 ActivityWatch） |
| 工时记录 | ✅ | 🟡 有追踪但没有 SP 详细 |

### 工具组合方案

```
Vikunja (任务后端) + Google 日历 (时间视图) + ActivityWatch (活动追踪)
                        ↑
                    Hermes (AI 大脑)
                        ↑
                    微信 (输入)
```
